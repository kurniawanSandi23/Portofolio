Proyek Data Science Basic

Proyek Data Science Basic adalah hasil dari Program Studi Independen (MSIB) yang berfokus pada penguasaan konsep fundamental Data Science, mulai dari pengolahan data hingga membangun model machine learning sederhana.

Fitur Utama

  • Pembersihan & preprocessing data
  • Eksplorasi data dengan Python
  • Visualisasi data interaktif
  • Model regresi & klasifikasi

Teknologi

  • Python (Pandas, NumPy, Scikit-learn, Matplotlib)
  • Google Colab
  • SQL untuk manajemen data
  • GitHub untuk kolaborasi

Hasil & Dampak

Proyek ini memperkuat pemahaman saya tentang alur kerja seorang Data Scientist: mulai dari data mentah hingga model prediksi. Hal ini mendukung saya untuk berkarir di bidang Data Analyst, Machine Learning Engineer, maupun Data Engineer.
