Proyek Smart Greenhouse

Smart Greenhouse adalah proyek otomasi berbasis IoT yang memantau dan mengatur suhu serta kelembaban di rumah kaca hidroponik secara real-time. Tujuannya adalah meningkatkan efisiensi pertanian modern dengan monitoring otomatis dan notifikasi kondisi abnormal.

Fitur Utama

  • Sensor suhu & kelembaban real-time
  • Otomasi kipas & pompa
  • Notifikasi WhatsApp saat kondisi tidak normal
  • Dashboard monitoring berbasis web

Teknologi

  • ESP32 + Sensor DHT11/DHT22
  • Python (Flask) untuk dashboard
  • Integrasi WhatsApp API
  • Database SQL

Hasil & Dampak

Proyek ini berhasil menjaga kondisi greenhouse tetap optimal, meningkatkan produktivitas tanaman sekaligus mengurangi kebutuhan tenaga manual. Solusi ini dapat diterapkan untuk pertanian modern skala kecil maupun besar.
