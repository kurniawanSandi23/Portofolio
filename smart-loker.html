Proyek Smart Loker

Smart Loker adalah proyek IoT untuk penyimpanan barang aman dengan autentikasi fingerprint. Solusi ini menggantikan kunci fisik tradisional dan mendukung monitoring real-time melalui aplikasi.

Fitur Utama

  • Autentikasi sidik jari
  • Database pengguna terintegrasi
  • Monitoring via aplikasi
  • Notifikasi real-time

Teknologi

  • ESP32 / Arduino
  • Sensor Fingerprint
  • MySQL Database
  • Aplikasi Web/Mobile

Hasil & Dampak

Sistem ini membuat penyimpanan barang lebih praktis dan terjamin keamanannya, cocok untuk kampus, kantor, dan fasilitas umum.
